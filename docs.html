Documentation made easy

Each machine function is shown as a scratch-style block with clear color coding. This page makes it fast to learn what every variable and action does.

Drill
Variable bool canMineWhether the drill can mine.
Variable int cooldownSeconds left before the drill can mine again.
Function mine()Collects a random ore or gem. After use, it cannot run again until 10 seconds have passed.
Conveyor
Variable string typeShow what its on the conveyor. Ore, Gem, Ore Bucket, Gem Bucket, Ingot, Weapon, Tool, Necklace, Bracelet, or Ring.
Variable bool isOccupiedTrue when an item is on the conveyor.
Function setSpeedspeedTurns the conveyor on or off and controls movement speed.
Furnace
Variable bool isFullTrue when the furnace cannot accept more items.
Function melt()Melts minerals or gems into a bucket of molten material.
Transformer
Variable bool isFullTrue when the transformer cannot accept more molten mineral.
Function transformtypeConverts enough molten mineral into a new item type: Ingot, Weapon, or Tool.
Jewelry Machine
Variable bool isFullTrue when the machine cannot accept more molten gem.
Function transformtypeConverts enough molten gem into Necklace, Bracelet, or Ring.
Dumping Truck
Variable bool isFullTrue when the truck is full of items.
Function sell()Sells every item in the truck and starts a 5 second cooldown before it can sell again.

How the blocks work

Think of each function as a Scratch-style block. The colors make it easy to scan the page and understand each machine at a glance.

Tip: Use the drill block to mine resources, then move items through conveyors. Melt them in the furnace, transform them into tools or jewelry, then sell with the Dumping truck.
